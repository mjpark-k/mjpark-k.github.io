  - 일반 버튼과 페이지네이션 버튼 2가지 컴포넌트
  - 각각 한개의 컴포넌트로 전체 페이지에서 활용

  - 기업리스트 클릭 시 해당 ID의 기업상세 페이지로 이동
  - 삭제 팝업 드롭다운

  - 투자내역이 없는 경우 텍스트 출력
  - 반응형 레이아웃 구현

  - 잘못된 url일 경우 NOT FOUND 페이지로 이동
